# Airpost

**Draw in the fog. Unveil. Post. Guess.**

Blind air-drawing game for Hack the Arts — MediaPipe hand tracking, depth ink, Intention Twin, memory flash, cinematic fog replay, and a guess-the-prompt wall.

## Local

```bash
npm install
npm run dev
```

Open the URL Vite prints. Allow camera (HTTPS or localhost).

```bash
npm run build
npm run preview
```

## Deploy

See **[DEPLOY.md](./DEPLOY.md)** for GitHub Pages steps.

## Stack

- TypeScript + Vite
- MediaPipe Hand Landmarker (on-device)
- perfect-freehand
- localStorage wall
